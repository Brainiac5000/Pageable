# Pageable

Pageable transforms a web page into a full page scrolling presentation.

  - Lightweight
  - Performant
  - Easy to set up
---
## Set up
### HTML
```html
<div id="container">
    <div data-anchor="Page 1"></div>
    <div data-anchor="Page 2"></div>
    <div data-anchor="Page 3"></div>
    <div data-anchor="Page 4"></div>
    ....
</div>
```
### JS
```javascript
new Pageable("#container");
```
